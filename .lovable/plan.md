# DreamFit — The incubator for the next Fit

## Status: Phase 1 MVP shipped

### What's built
- **Database**: `dream_tools`, `dream_backers`, `dream_comments` tables with RLS + auto-count triggers + trending score
- **Sidebar nav**: DreamFit added as equal-weight tool with ✨ Sparkles icon
- **Route**: `/DreamFit` integrated into Index tab system
- **Composer**: 3-field form (title, frustration, transformation) + type toggle (Feature/New Fit) + target Fit selector
- **Feed**: Sort by Trending/Most Backed/Fresh/Built, filter by type
- **Dream Cards**: Status badges, backer count, milestone labels, comment toggle
- **Comments**: Inline threaded comments per dream
- **Backing**: Heart-pulse "Back This" with auto-follow semantics
- **Status system**: Seeding → Momentum → Under Review → In Development → Live → Not a Fit

### Not yet built
- Admin status management panel
- "Born in DreamFit" badge on shipped features
- Weekly recap automation
- Share link / OG card generation
- Backer notifications on status change
